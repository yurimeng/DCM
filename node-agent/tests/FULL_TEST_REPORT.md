```json
{
  "job_id": "job_af1644a7",
  "match_id": "match_5c3f663e",
  "status": "settled",
  "locked_amount": 4e-08,
  "spent_amount": 3e-08,
  "actual_cost": 3e-08,
  "platform_fee": 1.5e-09,
  "node_earn": 2.85e-08,
  "refund_amount": 1e-08,
  "actual_tokens": 30
}
```

### 计算验证
```
locked = 0.001 × (5 + 30) / 1M = 4e-08 ✅
actual = 0.001 × 30 / 1M = 3e-08 ✅
platform = 3e-08 × 5% = 1.5e-09 ✅
node_earn = 3e-08 × 95% = 2.85e-08 ✅
refund = 4e-08 - 3e-08 = 1e-08 ✅
```

## 5. 网络流程

```
┌─────────────────────────────────────────────────────────┐
│  Prompt 下发 (优先协议)                                  │
│  Relay → QUIC → P2P → HTTPS (降级)                     │
│  日志显示: "Prompt via: relay | 网络: https"              │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  Job 提交 & 结算                                         │
│  强制使用 HTTPS                                          │
└─────────────────────────────────────────────────────────┘
```

## 6. 重试机制

```
无法连接服务器:
  ↓
  等待 30 秒
  ↓
  显示: "30 秒后重试... (1)"
  ↓
  重试连接
  ↓
  成功 → 继续
  失败 → 继续等待 (无限重试)
```
